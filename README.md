# FaceForm Free Content Bot

Полностью бесплатный автопостер без OpenAI API, платных нейросетей и Railway.

Каждый день GitHub Actions:

1. читает тематические RSS/Atom-ленты;
2. выбирает свежую релевантную публикацию;
3. формирует оригинальный безопасный пост по редакционным шаблонам;
4. создаёт фирменную карточку FaceForm;
5. публикует её в Telegram-канал;
6. сохраняет историю тем в репозитории, чтобы уменьшить повторы.

## Ограничение бесплатной версии

Это не языковая модель. Бот понимает тему по ключевым словам и создаёт полезный редакционный разбор по шаблонам. Он не пересказывает статью глубоко и не копирует её текст. В каждом посте остаётся ссылка на источник.

## Быстрый запуск через GitHub Actions

### 1. Создайте отдельного Telegram-бота

Через `@BotFather` создайте редакционного бота и добавьте его администратором канала с правом публикации сообщений.

### 2. Создайте GitHub-репозиторий

Загрузите все файлы проекта в корень репозитория.

Для полностью бесплатных GitHub Actions удобнее сделать репозиторий публичным. В публичных репозиториях стандартные GitHub-hosted runners бесплатны. В приватных репозиториях GitHub Free предоставляет включённую месячную квоту минут.

### 3. Добавьте секреты

Откройте:

```text
GitHub → Settings → Secrets and variables → Actions → New repository secret
```

Создайте четыре секрета:

```text
TELEGRAM_BOT_TOKEN      токен отдельного редакционного бота
TELEGRAM_CHANNEL_ID     @username_канала
FACEFORM_BOT_URL        https://t.me/username_основного_бота?startapp
CHANNEL_USERNAME        @username_канала
```

### 4. Выполните ручной тест

Откройте:

```text
GitHub → Actions → FaceForm Daily Post → Run workflow
```

После успешного теста workflow будет запускаться каждый день в 12:17 по московскому времени.

Время можно изменить в `.github/workflows/daily-post.yml`:

```yaml
schedule:
  - cron: "17 12 * * *"
    timezone: "Europe/Moscow"
```

Минуты выбраны не на начале часа, чтобы уменьшить вероятность задержки очереди.

## RSS-источники

По умолчанию используются тематические RSS-поиски по направлениям:

- мода и стиль;
- причёски и волосы;
- уход и beauty;
- мужской груминг;
- beauty-tech.

Свои ленты можно передать через `RSS_FEEDS` через запятую или с новой строки.

## Безопасность

Фильтр исключает темы о:

- похудении и экстремальных диетах;
- операциях, инъекциях и препаратах;
- диагнозах и лечении;
- токсичном сравнении внешности;
- трагедиях, слухах и скандалах.

Если свежая безопасная тема не найдена, бот публикует один из собственных нейтральных материалов FaceForm.

## Railway

Railway для этой версии не нужен. При желании проект всё ещё можно запускать как Cron-сервис через имеющиеся `Dockerfile` и `railway.json`, но это уже зависит от доступного тарифа Railway.
